## Feature
- 노션에 특정 템플릿으로 블로그 DB를 만들어 작성하면
- 해당 내용을 가져와 notion-to-md 로 변환
- md to jekyll-theme-chirpy html 로 변환
- github.io page 로 배포

## I Add More
- 노션에 embed 한 유튜브 영상 제대로 나오게 수정 
- workflow 에 오래된 action 들 최신 버전으로 변경

## Result
